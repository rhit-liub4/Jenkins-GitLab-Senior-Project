Problem Statement

Our project works with an integration mechanism between Jenkins and GitLab for GE Healthcare and their DevOps team. Jenkins is an open-source build server utilized by multiple teams within GE HealthCare, and Git is one of the most popular open-source distributed Source Code Managers (SCM). GE Healthcare is a company that provides digital infrastructure, data analytics & decision support tools helps in diagnosis, treatment, and monitoring of patients.​

Background

The GitLab Branch Source plugin is a primarily Java-based plugin that has 83 forks, 115 stars, and 59 contributors. The plugin also has 116 documented bugs, 55 of which have been closed. To gain experience with making and merging changes to this open-source project, we decided to first solve some existing bugs, like Issue #264, which was a null-pointer error with archived projects. ​
